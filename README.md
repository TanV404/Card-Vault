# 📇 CardVault - Business Card Scanner

> A privacy-first, zero-cost business card scanner and organizer built for mobile devices.

CardVault allows users to scan business cards using their phone camera, extract details using on-device OCR (Tesseract.js), and save them to a cloud database. It supports vCard export for instant contact saving and Excel export for bulk management.

## ✨ Features

* **📱 Mobile First Design:** Optimized for touchscreens with a sleek **Dark Mode** UI.
* **📷 Camera & Gallery Support:** Snap a photo instantly or upload from your library.
* **🧠 On-Device OCR:** Uses `Tesseract.js` to read text directly in the browser. **No expensive API keys required.**
* **🔄 Front & Back Scanning:** Intelligently merges data from both sides of a card.
* **👤 vCard Export:** One-tap "Add to Contacts" (.vcf generation).
* **📊 Excel Export:** Download your entire database as a spreadsheet (.xlsx).
* **☁️ Cloud Storage:** Securely stores data in Supabase (PostgreSQL).
* **✅ Data Validation:** Prevents saving empty or invalid contact details.

## 🛠️ Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Database:** [Supabase](https://supabase.com/)
* **OCR Engine:** [Tesseract.js](https://tesseract.projectnaptha.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Utilities:** `xlsx` (SheetJS)

## 🚀 Getting Started

### Prerequisites
* Node.js 18+ installed.
* A free [Supabase](https://supabase.com/) account.

### 1. Install dependencies
```bash
npm install
```

### 2. Setup Environment Variables
```bash
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```
### 3. Setup Database (Supabase)
* Go to your Supabase Dashboard -> SQL Editor.
* Run the following SQL query to create the table and allow public access (for MVP):
```bash
-- Create the table
create table cards (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  person_name text,
  company_name text,
  designation text,
  email text,
  phone text,
  address text
);

-- Enable Row Level Security
alter table cards enable row level security;

-- Create a public policy (Allow everyone to read/write for MVP)
create policy "Public Access" 
on cards
for all 
using (true) 
with check (true);
```

### 4. Run the development server
```bash
npm run dev
```
* Open http://localhost:3000 with your browser.
